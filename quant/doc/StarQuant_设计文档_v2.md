# StarQuant 量化交易 App 设计文档

> 版本: 2.0
> 更新日期: 2024-06-29
> 设计风格: Apple-clean + TRAE Work 品牌色
> 总页面数: 6

---

## 一、设计规范

### 1.1 色彩体系

基于 TRAE Work 设计系统，结合金融场景定制。

#### 品牌色

| Token | 色值 | 用途 |
|-------|------|------|
| Primary | `#4b3fe3` | 主按钮、Tab 指示器、选中态、FAB |
| Primary Hover | `#6a6fff` | 按钮悬停 |
| Primary Active | `#3f31c6` | 按钮按下 |
| Primary Soft | `rgba(171,183,255,0.36)` | 选中背景、标签底色 |

#### 功能色

| 用途 | 色值 | 说明 |
|------|------|------|
| 涨 (买入/盈利) | `#E53935` | A 股惯例，红色代表上涨 |
| 跌 (卖出/亏损) | `#43A047` | A 股惯例，绿色代表下跌 |
| 平 | `#9E9E9E` | 无涨跌 |
| 今日收益 | `#FF8A00` | 琥珀色，与涨跌区分 |
| 警告/止损 | `#C62828` | 深红，破坏性操作 |
| 成功/止盈 | `#2E7D32` | 深绿 |

#### 中性色

| Token | 色值 | 用途 |
|-------|------|------|
| Background | `#f2f2f7` | 页面背景 (苹果灰) |
| Surface | `#ffffff` | 卡片背景 |
| Border | `#e5e5e5` | 1px 分割线/卡片边框 |
| Text Primary | `#1d1d1f` | 标题、主要文字 |
| Text Secondary | `#6e6e73` | 副标题、辅助文字 |
| Text Tertiary | `#8e8e93` | 占位符、标签 |
| Icon Primary | `#1d1d1f` | 活动图标 |
| Icon Secondary | `#8e8e93` | 非活动图标 |

#### 图表色

| 指标 | 色值 | 用途 |
|------|------|------|
| MA5 | `#FFA000` | 5 日均线 |
| MA10 | `#E040FB` | 10 日均线 |
| MA20 | `#29B6F6` | 20 日均线 |
| 成交量涨 | `rgba(229,57,53,0.6)` | 上涨成交量 |
| 成交量跌 | `rgba(67,160,71,0.6)` | 下跌成交量 |

### 1.2 字体规范

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| Display | 28sp | Bold | 36sp | 总资产金额 |
| Headline L | 22sp | Bold | 28sp | 页面标题 |
| Headline M | 18sp | Bold | 24sp | 卡片标题 |
| Title L | 18sp | SemiBold | 24sp | 热股榜标题 |
| Title M | 15sp | SemiBold | 20sp | 段落标题 |
| Title S | 13sp | SemiBold | 18sp | 列表项标题 |
| Body L | 16sp | Normal | 24sp | 正文 |
| Body M | 14sp | Normal | 20sp | 列表详情 |
| Body S | 12sp | Normal | 16sp | 辅助信息 |
| Label L | 14sp | Medium | 20sp | 按钮文字 |
| Label M | 12sp | Medium | 16sp | 标签、Chip |
| Label S | 11sp | Medium | 16sp | 时间戳、极小文字 |

### 1.3 间距与圆角

| 组件 | 圆角 | 间距 |
|------|------|------|
| 页面边距 | - | 16dp 水平 |
| 卡片 | 16dp | 12dp 垂直间距 |
| 按钮 | 14dp | 14dp 内边距 |
| 输入框 | 12dp | 12dp 内边距 |
| Chip/标签 | 20dp (pill) | 6dp 间距 |
| 底部导航 | - | 无边距 |
| 头像 | 50% (circle) | - |
| 排名徽章 | 8dp (方形) | - |
| 价格涨跌标签 | 6dp | 8dp 水平, 2dp 垂直 |

### 1.4 阴影规则

遵循苹果设计: **几乎不用阴影**。
- 卡片: 无阴影，仅 1px `#e5e5e5` 边框
- 浮起元素 (FAB/BottomSheet): `shadow-md` (0 4px 12px rgba(0,0,0,0.08))
- 弹窗: `shadow-lg` (0 8px 24px rgba(0,0,0,0.12))

## 二、页面结构总览

### 2.1 信息架构

```
StarQuant
├── Tab 1: 首页 (Home)
│   ├── 资产概览卡片
│   ├── 热股榜 (Top 5 涨幅排名)
│   └── 热点榜 (新闻 + 关联个股)
│
├── Tab 2: 自选 (Favorites) ★新增
│   ├── 搜索添加股票
│   └── 自选股列表 (实时行情)
│
├── Tab 3: 行情 (Market)
│   ├── 搜索栏 + 收藏星标
│   ├── 行情详情卡片
│   ├── 分时走势图
│   └── K 线图 + MA 工具栏
│
├── Tab 4: 选股 (Screener)
│   ├── 预设策略 Chips
│   ├── 筛选条件摘要
│   ├── 开始选股按钮
│   ├── 筛选结果列表
│   └── 个股操作 (查看行情/加入自选/创建策略)
│
├── Tab 5: 策略 (Strategy)
│   ├── Tab: 运行中 / 策略配置
│   ├── 运行策略列表 (停止按钮)
│   ├── FAB: 新建策略
│   └── 启动确认弹窗
│
└── Tab 6: 我的 (Account)
    ├── 用户信息
    ├── 资产明细 (可用/持仓/盈亏)
    ├── 持仓列表
    ├── 交易记录
    └── 设置入口
```

### 2.2 底部导航栏

6 个 Tab，图标 + 文字标签:

| 序号 | Tab | 图标 | 选中态 |
|------|-----|------|--------|
| 1 | 首页 | house-filled | 品牌色图标 + 品牌色文字 |
| 2 | 自选 | star | 同上 |
| 3 | 行情 | candlestick | 同上 |
| 4 | 选股 | funnel/filter | 同上 |
| 5 | 策略 | gear/sliders | 同上 |
| 6 | 我的 | person | 同上 |

非选中态: 灰色 `#8e8e93` 图标 + 灰色文字
导航栏背景: `#ffffff`，顶部 1px `#e5e5e5` 分割线

### 4.2 关键联动规则

| 触发 | 行为 | 目标 |
|------|------|------|
| 首页热股榜点击 | 跳转行情页 + 查询该股票 | 行情页 |
| 首页热点榜点击 | Webview 展示新闻详情 | 内嵌 Webview |
| 自选条目点击 | 跳转行情页 + 查询 | 行情页 |
| 行情页星标点击 | 加入/移除自选 | 自选页同步更新 |
| 选股结果"查看行情" | 跳转行情页 + 预填代码 | 行情页 |
| 选股结果"加入自选" | 添加到自选列表 | 自选页同步更新 |
| 选股结果"创建策略" | 跳转策略页 + 预填代码 | 策略页 |
| 策略产生信号 | 生成交易记录 + 更新持仓 | 我的页同步 |

## 五、改良项 vs 原版对比

| 改良项 | 原版问题 | 新版方案 |
|--------|----------|----------|
| 新增自选 Tab | 无独立自选管理 | 新增第 2 个 Tab，搜索添加/左滑删除 |
| 首页改为热股+热点 | 自选假数据，无内容价值 | 热股榜 Top5 涨幅 + 热点新闻 |
| 去掉首页"可用"字段 | 参考图标注"移除" | 仅保留"今日收益"+"总盈亏" |
| 预设 Chip 选中态 | `selected=false` 写死 | 品牌色 tint + 动态 `selectedPreset` |
| 筛选条件摘要 | 无任何条件展示 | Chip 下方展示当前条件 + "清除" |
| 选股结果可操作 | onClick 空实现 | BottomSheet: 查看行情/加入自选/创建策略 |
| 行情星标同步 | 纯 UI 状态 | ViewModel toggleFavorite，同步自选页 |
| 多策略独立 Job | 共享单一 Job | StrategyRuntime.job 独立字段 |
| 重置账户确认 | 无确认直接重置 | AlertDialog 二次确认 |
| 启动策略弹窗 | 只显示类型+标的 | 显示完整参数摘要 |
